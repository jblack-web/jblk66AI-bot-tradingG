# jblk66AI-bot-tradingG

Advanced AI Trading Bot with Real-time Signals, Wallet Management, Futures Trading, Bitcoin Staking & Admin Dashboard.

## 🚀 Deploy (one command)

```bash
# 1. Copy environment config and fill in values
cp .env.example .env

# 2. Build frontend + start server
npm run deploy
```

The server runs on **port 3000** and serves both the API and the React admin dashboard.

## 🛠 Development

```bash
# Install all dependencies
npm run install:all

# Start backend only (requires frontend already built once)
npm start

# Build frontend
npm run build

# Rebuild and start
npm run deploy
```

## 🐳 Docker

```bash
docker build -t jblk66ai .
docker run -p 3000:3000 --env-file .env jblk66ai
```

## 📁 Project Structure

```
backend/          # Express API server (serves frontend + all API routes)
  ├── controllers/  # Route handlers (auth, admin, payments, staking, ...)
  ├── models/       # Mongoose models (User, StakingPool, Template, ...)
  ├── routes/       # Express routers
  ├── middleware/   # Auth middleware
  ├── utils/        # Helpers (sanitize, priceConfig, seedPools)
  ├── config/       # Cron jobs
  ├── seed/         # Template seeder
  └── public/       # Built React frontend (generated by npm run build)
frontend/         # React admin dashboard (source)
.env.example      # Environment variable reference
Dockerfile        # Multi-stage Docker build
```

## 🔑 Environment Variables

| Variable | Description | Default |
|---|---|---|
| `PORT` | Server port | `3000` |
| `MONGO_URI` | MongoDB connection string | `mongodb://localhost:27017/jblk66ai-trading` |
| `JWT_SECRET` | JWT signing secret | *(change in production)* |
| `NODE_ENV` | Environment | `production` |
| `BTC_USD_PRICE` | BTC/USD price for staking calculations | `50000` |

## 📡 API Routes

| Prefix | Description |
|---|---|
| `/api/auth` | Authentication (login, register, me) |
| `/api/admin` | Admin dashboard, users, tiers, promo codes |
| `/api/admin/staking` | Staking admin (pools, stakers, payouts, analytics) |
| `/api/staking` | User staking (stake, claim, withdraw) |
| `/api/payments` | Payments & deposits |
| `/api/withdrawals` | Withdrawal settings & requests |
| `/api/templates` | Template library |
| `/api/tiers` | Tier packages |
| `/api/trading` | Automated trading |
| `/api/ai-insights` | AI market insights |
